API Documentation

Endpoints:

GET /api/list_collections

Returns a list of all collections in the database.

                
["collection1", "collection2", ...]

GET /api/read/<document_id>

Returns a single document by its ObjectId from the "movies" collection.

                
{
    "_id": "ObjectId",
    "field1": "value1",
    "field2": "value2",
    ...
}

GET /api/read_all

Returns all documents from the "movies" collection.

                
[
    {
        "_id": "ObjectId1",
        "field1": "value1",
        "field2": "value2",
        ...
    },
    {
        "_id": "ObjectId2",
        "field1": "valueA",
        "field2": "valueB",
        ...
    },
    ...
]
